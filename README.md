# Lunch Hero

A Slackbot that helps you raise money for charity by getting lunch for your co-workers.

### Installation

Make sure you use `node` 8+ with `nvm`:

```bash
nvm use
```

Then install the dependencies:

```bash
yarn
```

### Usage

```bash
yarn dev
```

### Deployment (staging)

```bash
now
```
